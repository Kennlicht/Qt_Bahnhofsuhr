Eine Bahnhofsuhr, wie sie bei der DB eingesetzt wird. Achtet auf den Sekundenzeiger, wenn er bei der Zwölf vorbei kommt!

Ein bisschen abgeguckt bei spdrs60 (https://spdrs60.sourceforge.net/) und bei Qt's 'aclock' Beispiel

Compilieren mit
  setqt
  qmake
  make
DLLs mit Dependencies ermitteln und dazu kopieren
